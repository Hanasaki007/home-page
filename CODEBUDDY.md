# CODEBUDDY.md This file provides guidance to CodeBuddy when working with code in this repository.

## 项目概述
这是一个Chrome浏览器扩展，用于替换默认新标签页，提供毛玻璃风格界面。使用Manifest V3，纯前端实现，无构建系统。

## 常用命令
- **安装扩展**：在Chrome浏览器中打开 `chrome://extensions/`，开启开发者模式，点击"加载已解压的扩展程序"，选择项目文件夹
- **重新加载扩展**：在扩展管理页面点击刷新按钮
- **调试扩展**：右键点击扩展图标，选择"检查弹出内容"或"检查背景页"

## 高层架构
项目采用单页面应用架构，所有功能集成在一个HTML页面中：

**核心文件结构：**
- `manifest.json`：Chrome扩展清单，定义权限、图标和新标签页覆盖
- `newtab.html`：主页面，包含所有UI组件和模态框
- `css/style.css`：毛玻璃风格样式，包含主题系统和高性能模式
- `js/script.js`：所有功能的单一JavaScript文件（约1500行）

**功能模块（均在script.js中）：**
1. **时钟系统**：实时时钟显示，支持问候语和日期格式化
2. **搜索功能**：多搜索引擎支持（必应、百度、谷歌、DuckDuckGo、搜狗），可切换
3. **书签管理**：快捷网址的增删改查，支持拖拽排序，带彩色首字母图标
4. **主题系统**：6套渐变主题（星空紫、海洋蓝、日落橙、森林绿、极光、暖阳）
5. **背景管理**：自定义背景图片上传和压缩，支持恢复默认
6. **待办事项**：侧边栏Todo面板，支持勾选完成与删除
7. **游戏模块**：
   - 五子棋：人机对弈，支持悔棋和计分
   - 贪吃蛇：方向键操控，带开始页面和最高分记录
8. **数据持久化**：通过`chrome.storage` API保存所有用户数据，支持导出/导入JSON

**设计模式：**
- 单一职责：每个功能模块有独立的函数组
- 异步存储：使用Promise封装`chrome.storage`操作
- 事件驱动：所有交互通过事件监听器处理
- 状态管理：每个模块维护自己的状态变量

**关键技术：**
- CSS变量系统：主题颜色和毛玻璃效果通过CSS自定义属性管理
- Canvas API：游戏图形渲染
- 拖放API：书签拖拽排序
- 文件API：背景图片上传和压缩
- 键盘事件：游戏控制和快捷键

**注意事项：**
- 项目无构建步骤，直接修改文件即可
- 所有JavaScript代码在单个IIFE中，避免全局污染
- 高性能模式通过CSS类切换，禁用毛玻璃效果和动画
- 适配移动端，有响应式设计
